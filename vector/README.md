The vector module provides simple implementations for 2D, 3D, and 4D vectors.
All of them are immutable.

* [VectorNd](shared/src/main/scala/com/github/kright/math/VectorNd.scala): common interface
* [Vector2d](shared/src/main/scala/com/github/kright/math/Vector2d.scala)
* [Vector3d](shared/src/main/scala/com/github/kright/math/Vector3d.scala)
* [Vector4d](shared/src/main/scala/com/github/kright/math/Vector4d.scala)
