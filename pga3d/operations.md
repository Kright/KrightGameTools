## Binary operations

Row = type of the left operand `a`, column = type of the right operand `b`,
cell = result type of `a.<op>(b)`. Class names omit the `Pga3d` prefix.
An empty cell means the method is not generated because the result is identically zero.
`Pga3dMultivector` combines only with `Pga3dMultivector`; the specialized classes combine
with each other, but not with `Pga3dMultivector`.
